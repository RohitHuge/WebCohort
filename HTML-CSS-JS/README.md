# HTML-CSS-JS Project

This folder contains the HTML, CSS, and JavaScript files for the project.

## How to Setup and Run

1.  **Clone or Download**: Ensure you have the project files on your local machine. You can find the specific folder here: [https://github.com/RohitHuge/WebCohort/tree/main/HTML-CSS-JS](https://github.com/RohitHuge/WebCohort/tree/main/HTML-CSS-JS)

2.  **Open in Browser**:
    *   Navigate to the folder where you saved the files.
    *   Locate the `index.html` file.
    *   Double-click on `index.html` to open it in your default web browser.

3.  **Using VS Code (Recommended)**:
    *   Open the folder in Visual Studio Code.
    *   Install the **Live Server** extension.
    *   Right-click on `index.html` and select "Open with Live Server". This will automatically reload the page when you make changes.

## Files
*   `index.html`: The main structure of the web page.
